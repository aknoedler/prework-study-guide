Prework Study Guide

HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • Semantic elements are named descriptively.
  • Child elements are nested inside of parent elements.
  • Every element needs an opening tag and a closing tag.
  • The paragraph element, p, displays a block of text.
  • ol displays an ordered (numbered) list.
  • ul displays an unordered (bullet points) list.
  • li is used to creat list items.
  • img displays an image.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • A style sheet can be used to apply style to all elements in a document.
  • Custom classes can be created to selectively apply styles.

Git

  • Always make sure you pull the most recent version of the main branch when starting a new branch
  • git status checks what branch you are currenlty in
  • git checkout lets you switch what branch you are in
